# Block-Breaker
This game is an adaptation of the game Block Breaker. Your goal is to break all the bricks using a ball and a paddle.

For visual illustration, see the JPEGs and screenshots from the game.

How to start the game:

You will need: 
pygame==2.6.1 

Then:
1. Open your favorite Python interpreter (for me VS Code).
2. Make sure you have downloaded all the requirements.
3. Put all the files on the same level so they can see each other (may vary on the interpreter)
4. Run the main Python file.
